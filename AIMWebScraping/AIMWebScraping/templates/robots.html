Robots.txt, or robots exclusion protocol, is a file provided by most websites that communicate to web crawlers/scraper what the site's bot policy is

  • StackOverflow
  • Google
  • IMDB
  • Brick Link
  • Brick Set

Common features

User-agent

Indicates the agents the following rules apply to. User-agent: * will typically be the rules that apply to your bot

Allow

The allow specifies which areas of the site are approved for bots. This is a non-standard protocol, most bots will use whichever is more specific based on url length between allow and disallow when there are conflicts

Disallow

The disallow specifies which areas of the site your bots are not allowed to go

Crawl-delay

This is a non-standard robots directive so it is up to interpretation, but this is used to communicate to the bot how the site expects bots to be throttled. The widely accepted interpretation is this protocol is the number of seconds between requests

Sitemap

Another non-standard protocol. This communicates to a given bot where to find the websites XML sitemap.

Host

This protocol is used when a given site has multiple mirrors to indicate the preferred domain for the bot to use
