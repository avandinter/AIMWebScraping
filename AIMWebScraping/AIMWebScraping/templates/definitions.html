Web Crawler/Spider


A bot that systematically navigate the internet with the intent of indexing pages and following links.

Selection Policy

States how a given bot evaluates the importance of a given page for the purpose of prioritization

Revisit Policy

States how frequently a given bot will go back to a previously indexed page to detect changes

Politeness Policy

States how the bot will avoid overloading pages or cause undue stress on a web server

Parallelization Policy

Defines how bots are distributed and coordinated

Web scraper


The process of extracting information such as images, pricing information, or text from web pages

Human Copy and Paste

The manual extraction of information from a page. This will sometimes be required if a page creates barriers against automated scraping

Text Pattern Matching

Using patterns to extract specific information from a web page, this is usually facilitated by regular expressions

HTTP Programming and parsing

Retrieving the HTTP response from a given url request and parsing that response to scrape content
